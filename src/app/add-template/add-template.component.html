Explanation

1. Add your cover template in the "Cover Letter"-tab. Mark the keywords which you have to change for the specific application. There is an example cover letter attached with marked keywords. Try the example, or remove it and add yours, and add some keywords.

2. Then click on the "Application"-tab, and copy-paste the work described here. There is an example job description attached here, copy-pasted from LinkedIn. Try this out, or remove it and add yours. We will find keywords in the job description which you can use in your cover letter.

3. When finished, press "Create PDF" at the bottom of the page to edit your cover letter.

Nota bene: This software only supports English, so if the job description is in another language, we recommend running it through Google Translate and then copy-paste it here.
